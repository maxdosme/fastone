Fastone系统架构
Fastone Cloud Compiler
  • 自动分析各个任务间的依赖关系，将当前传统应用转换成云原生应用的工作流
  • 转换过程自动进行，快速完成，近乎零成本的的提升性能 – 目前有生信客户使用案例，转换后运行时间只需要原来的1/10，云上运行成本减少70%
  • 目前支持生物信息领域的Shell/Python脚本
Fastone Virtual Cloud
  • 一个界面，轻松管理和监控本地和各个云环境的运行状态
  • 30分钟启动初始化速石本地集群
  • 数分钟内在云端搭建大数据应用分析平台超大规模计算资源池
  • 本地集群一键增加和删除节点
  • 云端可根据计算负载弹性扩展
  • 云端支持使用spot instance，确保spot被抢分析过程不中断，极大的降低成本
  • 一键系统升级(滚动式各node依次升级、硬件Firmware、软件核心组件)
